Detailed Description

Helper functions for FindStereogenicUnits (using automorphisms).

These functions determine if an automorphism permutation invert the configuration of stereocenters by exchanging equivalent neighbor atoms (i.e. neighbor atoms with the same topological symmetry class).

Note
: The molecule should be ordered by topological canonical labels.
