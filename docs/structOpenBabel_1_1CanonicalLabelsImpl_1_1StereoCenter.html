Detailed Description

Structure to store and compute stereo center parities. Currently for both tetrahedral and cistrans stereochemistry. Precomputing these StereoCenter objects is done for performance reasons since parities need to be computed for each canonical candidate labeling.

Index(es) for the stereo center. This is used to sort the stereo centers by label. Tetrahedral stereo centers have one index in this vector and cistrans stereocenters have both double bond atom indexes in this vector.
