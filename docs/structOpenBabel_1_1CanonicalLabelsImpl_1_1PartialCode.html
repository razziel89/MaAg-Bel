Detailed Description

std::vector<OBAtom*> OpenBabel::CanonicalLabelsImpl::PartialCode::atoms

The atoms in canonical order. An atom is added when a label is assigned to it. When all atoms are labeled, this vector will contain all atoms in the fragment.

◆ bonds

std::vector<OBBond*> OpenBabel::CanonicalLabelsImpl::PartialCode::bonds

The bonds in canonical order. Since the atoms are labeled from another atom ("from"), these are the bonds connecting the "from" atom with the next labeled atom. The FROM list forms a spanning tree that does not include the ring closure bonds. These ring closure bonds are added at the end of this vector, in the correct order, in the CompleteCode function.

◆ from

std::vector<unsigned short> OpenBabel::CanonicalLabelsImpl::PartialCode::from

The FROM part of the canonical code. This is a vector containing the label of the "from" atom for the atoms in canonical order. The initial atom of a bonded fragment is not included in this list since there is no "from" atom.
