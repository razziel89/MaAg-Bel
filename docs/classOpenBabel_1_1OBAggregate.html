Detailed Description

Aggregate Class.

An aggregate is an object consisting of several molecules (i.e., covalently bound entities) and, hence, is based on OBMol. The OBAggregate class is designed to store the additional information associated with an aggregate, to enable manipulations of single molecules in the aggregate, the creation of such aggregates, and determine which atoms belong to the same molecule.

A very simple steric filter without the need to use the steric filter class.
